# Spring Boot CRUD 게시판 프로젝트

Spring Boot 학습 초기 단계에서,  
기본 개념을 실제 코드로 적용해보기 위해 진행한 개인 CRUD 게시판 프로젝트입니다.

이후 팀 프로젝트에 참여하기 전,
Controller–Service–Repository 구조와
JPA 기반 데이터 흐름을 직접 구현하며 익히는 것을 목표로 진행했습니다.

## 프로젝트 목적
- Spring Boot 기반 MVC 구조 이해
- JPA를 활용한 기본적인 CRUD 구현 경험
- 요청–응답 흐름과 계층 분리 구조 학습

## 주요 기능
- 게시글 등록 / 조회 / 수정 / 삭제 (CRUD)
- Spring Data JPA 기반 Entity 및 Repository 구현
- H2 In-Memory DB를 활용한 테스트 환경 구성
- Thymeleaf를 활용한 게시글 화면 구현

## 기술 스택
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Thymeleaf
- Gradle

## 학습 포인트
- Controller, Service, Repository의 역할 분리를 실제 코드로 경험
- JPA를 통한 데이터 저장 및 조회 흐름 이해
- MVC 패턴 기반 웹 애플리케이션 구조에 대한 기초 확립


※ 본 프로젝트는 학습 목적의 개인 프로젝트입니다.
